## Project structure

### Client

React app for adding thoughts.

### Server

NodeJS server that simulates the thoughts store.

## Run the project

Requirements Node v14.18.1

    cd client
    npm install
    npm start
    cd server
    npm install
    npm start
    Navigate to http://localhost:3000/

## Run frontend test

For now it only tests that App component renders

    cd client; npm test
